# LGX Atlantic Hub

A booking and information website for LGX Atlantic Hub — a multi-service entertainment venue in Okitipupa offering gym services, VIP lounge, outdoor bar, mini mart, and event hosting.

Built with Next.js, Tailwind CSS, and Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
